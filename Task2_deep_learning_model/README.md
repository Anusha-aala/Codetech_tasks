
# Deep Learning Project for Image Classification

## Internship Task 2 - CodTech Data Science Internship

### Objective

Develop a Deep Learning model for image classification and evaluate its performance using a benchmark dataset.

### Dataset

Scikit-Learn Digits Dataset

### Project Description

This project implements a neural network model using the Multi-Layer Perceptron (MLP) algorithm to classify handwritten digit images.

### Steps Performed

1. Loaded the Digits dataset from Scikit-Learn.
2. Split the dataset into training and testing sets.
3. Built a Multi-Layer Perceptron (MLP) neural network.
4. Trained the model on the training data.
5. Evaluated the model using test data.
6. Visualized the training loss curve.
7. Saved the trained model for future use.

### Technologies Used

* Python
* Scikit-Learn
* NumPy
* Matplotlib
* Joblib

### Model Architecture

* Input Layer: 64 Features
* Hidden Layer: 100 Neurons
* Output Layer: 10 Classes (Digits 0-9)

### Results

* Successfully trained a neural network model.
* Achieved high classification accuracy on the test dataset.
* Generated a training loss visualization.

### Output Files

* deep_learning_model.ipynb
* digit_classifier.pkl
* training_loss.png
