# Data Pipeline Development

## Internship Task 1 - CodTech Data Science Internship

### Objective

Develop an ETL (Extract, Transform, Load) pipeline using Pandas and Scikit-Learn for data preprocessing and transformation.

### Dataset

IBM Telco Customer Churn Dataset

### Steps Performed

1. Loaded the dataset using Pandas.
2. Cleaned the data by removing unnecessary columns.
3. Converted data types and handled missing values.
4. Encoded the target variable.
5. Split the dataset into training and testing sets.
6. Identified numerical and categorical features.
7. Applied Standard Scaling to numerical features.
8. Applied One-Hot Encoding to categorical features.
9. Created a preprocessing pipeline using Scikit-Learn.
10. Transformed the dataset and saved the outputs.

### Tools Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Joblib

### Output Files

* transformed_data.csv
* churn_pipeline.pkl
