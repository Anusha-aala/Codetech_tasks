# End-to-End Data Science Project

## House Price Prediction using Machine Learning and Flask

### Objective

Develop a complete data science project from data collection and preprocessing to model deployment using Flask.

### Dataset

California Housing Dataset from Scikit-Learn.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Flask
* Joblib

### Workflow

1. Data Collection
2. Data Exploration
3. Data Preprocessing
4. Feature Scaling
5. Model Training using Linear Regression
6. Model Evaluation
7. Visualization
8. Model Deployment using Flask

### Model Performance

* Mean Squared Error (MSE): 0.5559
* R² Score: 0.5758

### Output Files

* house_price_model.pkl
* prediction_graph.png

### Deployment

A Flask web application was created to predict house prices based on user input features.
